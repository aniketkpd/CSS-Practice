# CSS Practice

A collection of my early CSS practice projects and experiments.

## Overview

This repository contains various small projects and demos I created while learning and practicing CSS. Each folder focuses on a specific concept or technique.

## Folders

- **Animation** — CSS animation experiments
- **Box model** — Understanding padding, margin, border, etc.
- **Grid** — CSS Grid layout practice
- **List styling** — Custom list designs and styling
- **Transform** — `transform` property examples (rotate, scale, skew, etc.)
- **Transition** — Hover effects and smooth transitions
- **min-height** — Exploring height and min-height behavior

## Technologies

- HTML
- CSS (Pure vanilla CSS)
